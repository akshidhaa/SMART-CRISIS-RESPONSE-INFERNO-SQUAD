# shared/types

Shared TypeScript interfaces and Zod schemas used by every app and service.

## Models

- `Facility` — type, tier, mesh capabilities
- `User` — role, designation, multi-facility linking
- `Incident` — facility-aware, playbook-linked
- `Alert` — multi-language, multi-channel
- `EvacuationRoute`
- `MeshEvent` — flagship cross-entity coordination
- `MeshSubscription`
- `Camera` — AI detection input
- `ZoneCheckIn`

Built by **Prompt 1.1**.
