# web-public

Zero-login, lightweight Next.js 14 PWA for walk-in visitors and bystanders. Entry point: QR code at venue entrance encodes `https://scr-mesh.app/v/{facilityId}`.

Must load in <2 s on slow 3G. Lighthouse perf > 90. Multi-language auto-detect (en, hi, ta, te, mr, bn).

Build from **Prompt 1.6**.
